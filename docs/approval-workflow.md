# Approval Workflow

## Decision model

This repository operates under CEO approval.

### Required approvals

CEO approval is required before:

- merging to `main`
- changing public brand language
- changing primary CTA
- adding or removing products from the public site
- changing hosting or domain routing
- publishing pricing or service packages

## Suggested pull request template

### Summary
What changed?

### Reason
Why is this change needed?

### User impact
What will visitors notice?

### Approval required
- [ ] CEO approval received
